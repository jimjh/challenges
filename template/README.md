A+B Template Problem [[1](http://poj.org/problem?id=1000)]
==================

Here we write the description to the problem:

e.g. "calculate a+b"

#####Input Format

Two integers a,b separated by a space

#####Constraints

0 <= a,b <= 10

#####Output Format

Output a+b

#####Sample Input

```
1 2
```

#####Sample Output

```
3
```
